# Cyber Secure Hub

## Project info

A hub for cybersecurity resources and tools.

## How to run locally

1. **Clone the repository**
2. **Install dependencies**
   ```sh
   npm install
   ```
3. **Start the development server**
   ```sh
   npm run dev
   ```

## Technologies used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase
